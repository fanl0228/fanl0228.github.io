- ## [博客链接](https://fanl0228.github.io/)

- ## Usage
### 编译命令： 
    jekyll serve --port 4001
    
    bundle exec jekyll serve --port 4001



### 文件内容
_posts 博客内容

_pages 其他需要生成的网页


_layouts 网页排版模板

_includes 被模板包含的HTML片段，可在_config.yml中修改位置

assets 辅助资源 css布局 js脚本 图片等

_data 动态数据

_sites 最终生成的静态网页

_config.yml 网站的一些配置信息

index.html 网站的入口
